## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/Nataliia95254/To-Do-App
```

2. Navigate to the project directory:
```bash
cd To-Do-App
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```
